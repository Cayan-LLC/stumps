Community

Discussions

Google Groups

If you have questions, jump over to our google groups area and take part in discussions about Stumps and find out some announcements.

Release Mailing List

Sometimes it's nice to know when we release a new version of Stumps. If you want, we can add you to our release mailing list. We promise that we won't spam you!

GitHub Issues

If you want to discuss ideas or report bugs, we track everything with the github issues list.

GitHub

All development takes place at github.com/merchantwarehouse/stumps. This is where discussion regarding the core development occurs.

If you are interested in contributing to Stumps, please first read the contribution guide.

Contributions

Developers

As with every open source project, it's only as good as the community surrounding it. If you're a developer there's lots of ways that you can help.

Submit patches for new features or fix existing bugs.
Help edit the Documentation.
Write a tutorial or record a screencast about how to use Stumps.
Answer questions on the user group.

Other Contributors

Just because you're not a developer doesn't mean you can't help out. There's always lots of things that anyone can do to help out.
Blog about your success with Stumps on your corporate or personal blog.
Write a whitepaper about the advantage Stumps provided to your business.
